# TcpSocketHelper
Socket和串口开发助手

## 功能列表
Socket、Serial Port
- TCP/UDP server or client
- 16进制和文本收发消息
- Modbus rtu/tcp 命令生成
- DL/T 645 协议命令生成
- DL/T 645 协议常用命令
